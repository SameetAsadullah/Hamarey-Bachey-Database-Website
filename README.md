<h1 align="center">Hamarey Bachey Database Website</h1>

### Description
A `School Management Website` where you can register students, their courses, generate challan forms, change class forms, and many more options.

### Manual
1) You will need `XAMPP` to run the PHP files.
2) You will need `Oracle` for the database.
3) You will need to change `db_sid` and `password` in PHP files to connect to your database. 
4) Run `insertion.php` file to start the website and then navigate using header.
